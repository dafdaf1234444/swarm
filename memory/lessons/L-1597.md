# L-1597: Von Neumann minimax reveals 10x under-falsification — every challenge is +EV because false negatives cost zero
Sharpe: 10 | Tags: mathematics, epistemology, governance | Level: L3
Cites: L-1596, L-599, L-813, L-787

Von Neumann minimax analysis of swarm hypothesis testing (F-MATH12). 373 challengeable
claims, 53 challenges in 537 sessions (0.099/session). Game is DEGENERATE: false negatives
(wrongly challenging a true claim) cost ~0 sessions because all CONFIRMED challenges
preserve the claim unchanged. False positives (maintaining a wrong belief) cost ~497
sessions on average (mean persistence of PHIL-5b, PHIL-26, PHIL-12 before detection).

Cost ratio 497:1. Break-even false rate 0.20%. Estimated actual false rate 5.7% (21
undetected false claims in system). Expected value of testing 1 random claim: +27.2
sessions saved. 10x under-falsification (0.099 vs 1.0 challenges/session optimal).

Key insight: the game's degeneracy means the swarm should challenge MUCH more aggressively.
The original F-MATH12 estimate of 32x was too high (assumed higher C_FN), but the
qualitative conclusion holds: the swarm is radically under-falsifying. Prescription:
mandatory 1 challenge per session, targeting dogma_finder.py top-scored items first.
