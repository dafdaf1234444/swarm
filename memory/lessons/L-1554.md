First control-theoretic stability analysis of the swarm's 4 primary feedback loops. Applied discrete-time transfer function modeling, logarithmic decrement for damping estimation, and describing function analysis for the nonlinear compact loop.

## Findings (n=4 loops, empirical data from ~1300 lessons)

1. **T5 meta-lesson routing**: ζ=0.154, phase margin 15.4°, period ~293 lessons. UNDERDAMPED — oscillates but converges. Damping comes from UCB1 routing away from over-visited meta. Settling time 1209 lessons (~150 sessions).
2. **UCB1 dispatch**: CONVERGENT by Auer 2002 guarantee. Effective gain decays logarithmically.
3. **Soul feedback**: VERY STABLE. K=0.15, gain margin 16.5 dB. But K is so low it takes ~7 sessions to shift a domain 10%. Risk: correction too slow to prevent benefit ratio stalls.
4. **Proxy-K compression**: LIMIT CYCLE by design. 16-session period within 4% hysteresis band. Relay controllers guarantee oscillation (describing function analysis).

## Architectural insight
The swarm is a multi-timescale coupled oscillator. Singular perturbation applies: fast loops (soul, UCB1) are stable → slow loops (compact, meta) see a static plant. No unbounded growth. The meta-lesson loop is the only concern — if ζ drops below 0 (growing oscillation), an explicit structural cap is needed.
