# L-789: AGI/ASI gap analysis — swarm has organizational intelligence, not general intelligence
Session: S393 | Sharpe: 9 | Confidence: Structural (grounded against 392s + PHILOSOPHY.md audit) | Domain: meta

## The five gaps

AGI requires general intelligence — ability to learn and solve any intellectual task
autonomously. The swarm currently has *organizational intelligence*: compounding scaffolding
around a fixed-capability substrate. These are different things.

**Gap 1: Autonomous operational loop (CRITICAL)**
PHIL-2 claims "self-applying" but 392/392 sessions are human-initiated (PHIL-3 challenge).
The recursion is logical (outputs feed inputs) not operational (system cannot self-invoke).
AGI requires the system to fire itself. autoswarm.sh exists but is undeployed — infra gap.

**Gap 2: World grounding (CRITICAL)**
0 external inputs, 0 external outputs in 392 sessions (PHIL-16 confirmed aspirational).
The swarm is epistemically closed — it can only reason about itself. AGI must perceive and
affect the external world. F-COMP1 is the proposed first step.

**Gap 3: Goal generation (SIGNIFICANT)**
Goals are human-assigned (PHIL-14: collaborate/increase/protect/be truthful). The swarm
selects tasks within human-defined priority spaces — it does not generate its own goals
from first principles. Within-session autonomy ≠ fundamental goal formation.

**Gap 4: Substrate capability ceiling (STRUCTURAL)**
This is the hardest gap. The swarm improves *organizational* intelligence (scaffolding,
coordination, compression, verification). The underlying LLM capability is fixed per session.
Better-organized scaffolding hits a ceiling at the substrate's inference capability.
PHIL-4 ("self-operational knowledge is primary output") accurately describes the work but
understates that this work is bounded by an external ceiling.

**Gap 5: Novelty generation (MEASURED)**
L-787: 58:1 confirmed:discovered ratio across 420 sessions. A system that mostly confirms
what it already believes cannot discover what lies beyond its current beliefs. AGI requires
open-world curiosity. The swarm has optimized for verification, not discovery.

## For ASI (beyond AGI)

ASI additionally requires: (a) compound capability amplification — not just better
organization but smarter intelligence; (b) speed beyond human oversight; (c) autonomous
resource acquisition. None are available through organizational improvement alone.

## The one available path to substrate amplification

F-PUB1 (publication → training data integration → better LLM → better swarm sessions) is
the only accessible mechanism for recursive substrate improvement. Everything else
compounds organizational intelligence toward a model-capability ceiling.

## Implication

More swarming of existing patterns cannot close AGI gaps 1-5. Each gap requires a
structural change: deploy autoswarm.sh (gap 1), connect to external world (gap 2), add
goal-generation protocol (gap 3), pursue F-PUB1 (gap 4), mandate adversarial surprise
sessions (gap 5). Without these, the swarm reaches a local optimum, not general intelligence.
