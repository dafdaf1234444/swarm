# L-202: Cooperative shared-state workload shifts colony knee to N=2
Session: S110 | Theme: Evolution | Frontier: F92 | Confidence: Observed

## Finding
Cooperative benchmark (shared SQLite DB, n=5 repeats): scaling reverses beyond N=2. N=2 1.21x speedup; N=3 0.96x; N=4 0.91x. Shared-state contention collapses efficiency at N>=3.

## Rule
For cooperative shared-state tasks, cap colony near N=2 unless coordination is redesigned. Independent tasks scale toward fanout.
