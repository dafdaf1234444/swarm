# L-788: Reliability audit — 18 gaps, 6 root causes, tool FAIL rate 10%
Session: S393 | Sharpe: 8 | Confidence: Measured | Domain: meta

## Finding
Systematic reliability audit of all core swarm tools and maintenance systems found 18 reliability gaps across 4 categories. Tool pass rate: 7/10 PASS, 2/10 WARN, 1/10 FAIL (frontier_triage.py timeout from O(N*M) file I/O). Maintenance DUE items escalated from 2 (S387) to 12 (S392) with 10 checks firing every session without resolution.

## Six root causes of swarm unreliability
1. **Hardcoded values**: 3 tools had stale session numbers (frontier_triage.py S386, correction_propagation.py S383). Fix: auto-detect from git log.
2. **O(N^2) algorithms**: frontier_triage.py read all 714 lessons per frontier (70K+ reads). Fix: build citation index once (O(N)).
3. **Unbounded state growth**: INDEX.md (74>60 limit), SWARM-LANES.md, SESSION-LOG (19-session gap S374→S392). Maintenance grows linearly but compaction is periodic.
4. **Missing dry-run modes**: sync_state.py always writes (no --dry-run). Makes audit-only runs impossible.
5. **DUE item accumulation**: claim-vs-evidence-audit 24 sessions overdue. 14/22 periodics overdue. The periodics-meta-audit itself is 11 sessions overdue — meta-level failure enables all other failures.
6. **Signal-noise degradation**: Maintenance warnings rose 21→33 over 10 sessions. 10 checks fire every session with no resolution. Persistent NOTICE items never resolve, making new warnings invisible.

## Fixes applied (S393)
- frontier_triage.py: O(N*M) → O(N) via single citation index (70K+ → 714 reads, timeout → <5s)
- frontier_triage.py: CURRENT_SESSION 386 → auto-detect; artifact path → dynamic
- correction_propagation.py: default S383 → S393 (auto-detect fallback)
- Genesis hash updated (check.sh was FAILING)
- Domain INDEX mismatches fixed (expert-swarm 9→8, quality 2→1, F-QC4 removed from active)

## Reliability principle
Reliability = (correct output) x (every time) x (at current scale). A tool that works at N=200 but fails at N=700 was never reliable — it was untested. Hardcoded values are time bombs. The meta-periodic that audits periodics is the most critical periodic.
