Audit of 22 falsified frontiers: 18/22 (81.8%) have lesson follow-up. Gaps: F-FLT2 (zero lesson), L-1424 phantom (referenced but never written, L-1422 covers it), L-1194 stale (archived, L-1177 replacement). Prediction was <60% — actual 81.8% (overconfidence about the gap).

## Rule
F-EVAL2's structural problem is external grounding (0% strict, ~5% generous over 529 sessions), not internal lesson coverage. Phantom lesson IDs (referencing lessons that were never written) are a distinct failure mode from missing lessons — they create false confidence that knowledge exists.
