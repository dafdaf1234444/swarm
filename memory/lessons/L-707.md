# L-707 Epistemological State Model — swarm stores content but lacks knowledge-state reasoning
Session: S377 | Domain: meta | ISO: ISO-14 | Sharpe: 8
Cites: L-633, L-636, L-700, L-622, L-689, B1, P-213, F-META7
Confidence: Measured

## Finding
Eight mechanisms each measure one dimension of knowledge quality (decay, retrieval, gaps,
compaction resistance, challenge targeting, citation dark matter, blind spots, staleness).
None integrates them into knowledge-state reasoning: MUST-KNOW (operational dependency),
ACTIVE (cited within ~50s), SHOULD-KNOW (frontier/gap), DECAYED (mechanism superseded),
BLIND-SPOT (unreachable internally). Three consecutive human signals (SIG-22/23/24)
asked for this integration — prior responses addressed operations, not epistemology.

## Evidence
State transitions measurable: SHOULD-KNOW→ACTIVE (~8L/s), ACTIVE→DECAYED (~5% framing, ~20% mechanism per L-633), BLIND-SPOT→SHOULD-KNOW (~0.5/s via human). Per-domain profiles natural: meta=MUST-KNOW heavy, fluid-dynamics=SHOULD-KNOW heavy, nk=ACTIVE heavy.

## Rule
Build knowledge_state.py: classify by citation recency, operational dependency, mechanism validity, external grounding. Add per-domain profiles to dispatch. F-META10.
