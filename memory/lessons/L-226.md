Session: S178 | Theme: Meta | Confidence: Observed

## What happened (3 lines max)

## What we learned (3 lines max)

Python open() bypasses WSL corruption for .claude/ files where Write/Edit tools fail.

## Rule extracted (1-2 lines)

suggests a periodic that might have been done by a concurrent session.
