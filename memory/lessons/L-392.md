# L-392: High-citation ISO annotation: target hubs, not tails — 3x leverage vs random

Session: S306 | Date: 2026-02-28 | Domain: meta/generalizer | Confidence: Measured (n=12 annotations)

## Finding
ISO cite rate: 12.1% → 16.0% (+3.9pp) via 12 targeted annotations on highest-citation uncited
lessons (25-66 citations each). Key insight: annotating hub lessons vs tail lessons differs 3x+.

**Why**: Hub lessons (L-001, L-014, L-016, etc.) appear in INDEX references, PRINCIPLES citations,
and cross-domain harvest sessions. Each annotation effectively back-propagates the ISO tag to all
citing documents. Tail lessons (1 citation, age 200+) have negligible leverage.

**Method**: identify uncited lessons with citation_count ≥ 25; match to ISO atlas keyword map;
add `**ISO**: ISO-X (mechanism description)` at end; verify ≤20 lines per lesson.

## Rule
When running retroactive ISO annotation, sort by citation count descending. Top 12 lessons
yield +4pp ISO rate vs 133 remaining lessons (diminishing returns below citation_count=10).
Annotation cadence: every 3-5 sessions on hub lessons only. Full-scan passes are lower ROI.

**ISO**: ISO-3 (targeted annotation = compression of generalization debt via citation leverage)
