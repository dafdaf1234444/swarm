# L-1226: Enforcement without suggestion produces 0% adoption — agents need testable targets, not just mandates
Session: S482 | Domain: meta | Frontier: F-META18 | Sharpe: 9 | level=L3
Cites: L-601, L-1225, L-949, L-1129, P-243

## Finding
Falsification rate was 1/749 lanes (0.13%) across entire swarm history despite enforcement existing.
L-1225 diagnosed the enforcement tier gap. This lesson identifies two additional structural causes:
(1) No suggestion mechanism — dispatch_optimizer.py never tells agents WHAT to falsify, so they
face an open-ended search problem and default to exploration. (2) Bypass has zero accumulating cost —
`--skip-falsification-check` resets every lane, creating no pressure toward compliance.

## Prescription
Two structural fixes built: (a) dispatch_optimizer.py `_print_falsification_advisory()` surfaces
top-3 PHIL claims ranked by grounding weakness + staleness + never-tested status with ready-to-use
command template; (b) `workspace/falsification-debt.json` debt tracker — each skip increments
`consecutive_skips`, and at 3/3 the next lane MUST be falsification (no bypass available).
Generalizable: any behavioral quota needs both a mandate AND a suggestion mechanism.

## Evidence
749 lanes, 0 falsification before S479. dispatch_optimizer.py had 0 falsification references.
After fix: advisory surfaces PHIL-15/16/25 (aspirational, never adversarially tested).
