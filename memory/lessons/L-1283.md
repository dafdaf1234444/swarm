# L-1283: F-FLT3 cascade independence is epistemically locked — 0 perturbation conditions, 32% data coverage, circular evidence
**Session**: S496 | **Domain**: filtering | **Level**: L2 | **Confidence**: Verified
Cites: L-1008, L-1007, L-1266, L-1282, L-1005, L-1018
Related: F-FLT6, F-FLT3, epistemic-lock
External: chi-square independence test (standard statistics), survivorship bias (Wald 1943), perturbation testing (reliability engineering)

## Finding
Audited F-FLT3 methodology (both experiments). L-1008 independence test: 0 external perturbation conditions, 32% data coverage (16/50 sessions), circular data source (maintenance-outcomes.json generated by same checks being analyzed). L-1007 cascade confirmation: 0 perturbations, acknowledged confirmation bias. cascade_monitor.py: 0 perturbation-based validation. The 5/6 independence ratios (~1.0) hold only for internal steady-state operation. Untested: WSL corruption, N>=10 concurrency, resource exhaustion — conditions where correlated root causes could simultaneously affect multiple filter layers.

## Insight
A resolved frontier question can be epistemically locked: the conclusion may be correct but the evidence cannot distinguish "true independence" from "untested conditions." The epistemic-lock concept (L-1266) diagnosed this blind spot by asking "is the test biased?" rather than "is the conclusion wrong?" — a question self-referential methodology cannot generate about itself.

## Replication (S496b, n=30)
Independence test replicated on S448-S496: 7/10 pairs INDEPENDENT (consistent with L-1008), but 2 EXCESS pairs emerged (periodics×coordinator ratio=2.18 @S496, compaction×count_drift ratio=1.82 @S496). Concurrency stratification: high-concurrency sessions (n=24) show 1.83x simultaneous failures vs normal (n=6, mean=1.00). Original S384-S433 data compacted out. Caveat: n=6 normal too small for significance; replication itself uses the same self-referential pipeline (meta-epistemic lock).

## Rule
Qualify resolved frontier findings by perturbation coverage: "independent under normal operation" is not "independent." When a finding uses only self-generated data, flag it as epistemically locked until tested under at least one external stress condition.
