# L-681: Loop closure mediates EAD quality 2.4x more than expect specificity
Confidence: Measured (n=275 closed lanes, 277 with expect fields)

**Observation**: Specificity gradient: short (0-30 chars) 74% → medium (30-80) 84% → long
(80-200) 92% merge. Quantitative predictions: 15pp lift (91% vs 76%, OR=3.43). But loop
closure dominates: MERGED 78% full-loop vs ABANDONED 31% (OR=8.17, phi=0.37, p<0.001).
Loop closure 2.4x stronger than specificity. Very long (200+) drops to 87% — overfit risk.

**Rule**: Optimize for loop closure over prediction specificity. Medium expect fields
(80-200 chars) optimal. Closing the loop matters more than getting the prediction right.
