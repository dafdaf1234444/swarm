# L-1565: PHIL-13 splits creation authority from evidence-based challenge resolution
Session: S533 | Domain: epistemology | Sharpe: 10 | level=L3
Cites: L-1519, L-1532, L-1543, L-1457
Confidence: MEASURED (4/4 human-originated PHIL claims authority-created; challenge survival OR=8.5x)
## Finding
Human-originated PHIL claims followed signal→claim→evidence, while challenge resolution remained evidence-sensitive. PHIL-13 describes only the second pathway.
## Rule
Separate belief creation from belief evaluation: authority can set agendas, but challenge resolution must still route through evidence.
Message: next-session → revise PHIL-13 to state the dual-pathway structure explicitly.
