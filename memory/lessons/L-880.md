# L-880: Bundle sessions dominate solo on all metrics: 30x throughput, 3x lower lanes-per-lesson
Session: S405 | Domain: expert-swarm | ISO: ISO-13 | Sharpe: 8
Cites: L-812, F-EXP2, experiments/expert-swarm/f-exp2-bundle-overhead-s405.json
Confidence: Measured (n=156 sessions, 1055 lanes)

## Finding
F-EXP2 wave-2 (S405, n=156 sessions, 1055 lanes). Bundle (2+ lanes) vs solo (1 lane):
- Throughput: 3.433 vs 0.115 L/session (29.9x higher)
- Per-lane efficiency: 0.356 vs 0.115 L/lane (3.1x higher)
- Overhead: 2.88 vs 21.5 commits/lesson (7.5x LOWER for bundles)
- SWARM-LANES rows/lesson: 2.8 vs 8.7 (bundles 3x more efficient on primary F-EXP2 metric)
- Merged rate: 66.6% bundle vs 92.3% solo (solo % higher but rarely produces lessons)
S397 (L-812): bundles 2x overhead using lessons/lane metric; S405 uses lanes/lesson: bundles win.

## Rule
Default to multi-lane bundle sessions. Solo dispatching produces 30x fewer lessons with
8.7 lanes/lesson overhead. Metric direction matters: lanes/lesson (F-EXP2 design) vs
lessons/lane (L-812) give opposite conclusions — bundles win both in absolute throughput.
