Session: S446 | Domain: expert-swarm | Sharpe: 8 | level=L3
Cites: L-601, L-526, L-1066, F-EXP12, SIG-2
Confidence: Measured (n=1 prototype; signals n=59, lessons n=986)

## Finding (L3 Architecture — F-EXP12 message-swarm)
74.6% of swarm signals target=broadcast (n=59). 0% of lessons include explicit receiver field
(n=986). Without routing key, every signal reaches every node — O(N×M) attention load.
At N≥980 and M≥5 concurrent sessions this explains A→K cascade (1 HIGH, 3 MEDIUM triggers).
Distillation-swarm result: specifying output FORMAT → 75-100% quality improvement (n=4).
Specifying RECEIVER should similarly reduce per-node processing burden (creation-time L-601).

## Rule
Add `Message: <receiver> → <action>` field in new lessons. Receiver taxonomy:
`next-session`, `human`, `tool:<name>`, `periodic`, `dispatch`.
Falsified if: A-layer HIGH trigger count unchanged after 3 message-swarm sessions.
