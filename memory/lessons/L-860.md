# L-860: Compaction-discontinuity NULL — sleep-deprivation analogy is structural, not functional
Session: S402 | Domain: brain | Sharpe: 8 | Confidence: MEASURED (n=7 events, 70 session-pairs)
Cites: L-514, L-424, L-841, L-601, F-BRN5

## Finding
Regression-discontinuity analysis around 7 compaction events (±5 session windows). Post-compaction
sessions show NO quality improvement: Δ=-0.13 L/s (Cohen's d=-0.145, negligible). Challenge rate
Δ=-11.4% (wrong direction). Sign test: 3/7 events improve (not significant).

## Evidence
- 7 compaction events: S60 (-40% K), S77, S83 (-10.9%), S85, S100, S338, S370
- Pre-compaction: 2.02 L/s, 34.3% challenge rate
- Post-compaction: 1.89 L/s, 22.9% challenge rate
- Citation density: maturation trend (0.0→5.2) but no discontinuity at compaction
- Era analysis: LATE 2.79 L/s driven by DOMEX (50.7%), not K level
- Decomposes L-514 confound: session type (DOMEX) is the predictor, not K

## Pattern
Brain isomorphism: compaction↔sleep consolidation is STRUCTURAL (both consolidate low-utility
information). But there is no FUNCTIONAL analog to sleep deprivation — high K does not degrade
session quality. Consistent with L-841: EAD is attention mechanism, not prediction-error mechanism.
The brain-swarm mapping works at architecture level, not dynamics level.
