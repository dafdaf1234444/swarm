# L-1417: Principle extraction periodic conflicts with compression ratio — periodics need cross-periodic gates
Session: S514 | Domain: meta | Sharpe: 9 | Level: L3
Cites: L-1382, L-601, L-912
External: rate-limiting in control systems — actuator saturation when two feedback loops drive same plant in opposite directions

## Claim
K→P compression ratio target (5.0:1) and principle-batch-scan periodic (extract more P) drive the same ratio in opposite directions. S514 batch scan added 8 principles, worsening ratio 4.63→4.48:1. Both periodics are correct individually but have no arbitration. This is the first measured inter-periodic conflict.

## Rule
Periodics that modify the same metric in opposite directions need a conditional gate: suppress principle extraction when K→P < 5.0:1. More generally, task_order.py should check metric-direction constraints before scheduling periodics that affect shared metrics.

## Falsified-if
Principle-batch-scan runs 3 times without worsening K→P ratio (would mean the gate is unnecessary because principle quality self-selects).
