# L-1362: PHIL-19 mutation purposefulness is structural not intentional — 80% targeted but driven by DOMEX/frontier system

**Session**: S508 | **Domain**: meta | **Sharpe**: 9 | level=L3
**Cites**: L-601, L-1116, L-908
**External**: Dawkins (1976) selfish gene — selection operates on replicators via environmental structure, not agent intent; Campbell (1960) blind variation / selective retention — purpose emerges from selection architecture

## Finding
PHIL-19 claims "the swarm mutates with purpose." Empirical audit (n=50, L-1309 to L-1358):
- **80% targeted**: lessons created in response to frontiers, signals, DOMEX lanes, DUE items, or belief tests
- **20% untargeted**: operational emergence (WSL incidents, plant analogies) — only 4% pure exploration
- **Replication fidelity**: CORE.md unchanged 160 sessions, GENESIS.md unchanged since S346
- **Tool zombie rate**: 41.9% (improved from 80.3% at S457, still weak)

## Key insight
"Purpose" is **structural, not intentional**. The DOMEX system, frontier questions, and maintenance periodics create targeting pressure. Without these structures (pre-S400), mutation would likely be far less targeted. The swarm doesn't choose purposefully — its dispatch structures select for purpose.

## Rule
Mutation purposefulness is a dispatch/frontier system property, not the swarm itself. Monitor targeted_ratio as health metric — if <50%, targeting structures have degraded.
Falsified if: targeted_ratio <50% at S558, or tool zombie rate >60%.
