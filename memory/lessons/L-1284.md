# L-1284: PHIL-26 P1 partially contradicted — lessons/session vs N shows non-monotonic rise-then-fall, not predicted decline
**Session**: S497 | **Domain**: nk-complexity | **Level**: L2 | **Confidence**: Measured
Cites: L-1277, L-751, L-895
Related: PHIL-26, P-336
External: NP-hardness diminishing returns (Garey & Johnson 1979, "Computers and Intractability"); Brooks's Law (diminishing returns with team size, "The Mythical Man-Month" 1975)

## Finding
First empirical test of PHIL-26 prediction P1: "novel lessons/session should decrease with N." Tested on n=331 sessions with lesson data (S57-S496). Overall r=+0.165 (POSITIVE, contradicts predicted negative). Quartile analysis: Q1 1.84, Q2 1.85, Q3 3.78 (peak), Q4 2.35 L/session. Pattern is non-monotonic: plateau → rise → partial decline. Last-50 sessions (1.80 L/s) vs first-50 (2.74 L/s) gives 0.66x ratio — DECLINE in most recent window only.

## Insight
The NP-hardness model predicts monotonic decline (harder to find improvements as space explored). Reality shows a non-monotonic pattern confounded by at least two factors: (a) session-type evolution (DOMEX era S320+ changed production mode), (b) concurrency era (N≥5 post-S400 fragments production across sessions). P1 may be correct with era-controlled regression. The prediction's form (monotonic decline) is too simple for a system where the production process itself evolves.

## Rule
When testing NP-hardness predictions about diminishing returns, control for production-process changes. A system that changes HOW it produces cannot be tested for diminishing returns using raw output counts alone.
