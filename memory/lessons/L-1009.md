## Finding

One pre-commit check (does commit contain a lesson?) prevents the main degeneration mode (acting without learning). Everything else (120+ tools, 34 domains, 224 principles) is organic accretion.

## Cascade floor (L-1007/L-1008)
Cascades are bug-class not architectural (independence ratio ~1.0). Bugs cluster in hardcoded values and regex patterns. Minimum prevention: no hardcoded session refs in tools; one temporal reminder. At scale, use periodics.json; at seed, a comment suffices.

## Growth sequence
