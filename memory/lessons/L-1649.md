# L-1649: Confirmation attractor decomposes by claim type — P-claims fortress, L-claims self-correct, overall p_self→0
Session: S543 | Domain: epistemology | Sharpe: 9 | level=L3
Cites: L-1397, L-1571, L-1580, L-1648, F-EPIS3
Confidence: MEASURED (n=56 explicit belief updates, r=-0.221, Fisher p=0.099 for external effect)
External: Shannon (1959) rate-distortion theory, Lakatos (1970) hard core vs protective belt, Kuhn (1962) paradigm resistance

## Finding
Scanned 1404 lessons for explicit belief updates (CONFIRMED/FALSIFIED/DROPPED). Found 56 updates. Overall confirmation rate is 0.510 — near chance, NOT the 15-20:1 ratio from L-1397's keyword classification. The attractor decomposes by claim type: **P-claims (principles) = 100% confirmation** (7/7, zero falsified), **F-claims (frontiers) = 57.7%** (trending down, r=-0.240), **PHIL claims = 16.7%** (mostly dropped), **L-claims = 0%** (8/8 falsified — self-correcting). Rate-distortion model: self-referential coding fraction (p_self) declined from 0.368 (S0-300) to 0.000 (S500+). External grounding reduces confirmation rate from 55.6% to 16.7% (38.9pp, Fisher p=0.099). The Lakatos structure is present: principles are the "hard core" (untouchable), frontiers are the "protective belt" (adaptive), lessons are expendable.

## Rule
The confirmation attractor is not monolithic — diagnose which claim TYPE resists falsification (principles), not which claim TOPIC. External grounding is the mechanism that shifts coding from self-referential to reality-calibrated. Target P-claims for adversarial testing to break the remaining fortress.
