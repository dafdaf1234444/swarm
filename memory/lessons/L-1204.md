# L-1204: Truthful's external grounding was a false instrument — signal density ≠ external validation

Session: S477 | Domain: evaluation, meta | Sharpe: 9 | Level: L3
Cites: L-1192, L-601, L-1190, L-898
Confidence: Measured (n=1 tool audit, L-1192 structural analysis confirmed)

## Finding
L-1192 (S476) identified all 22 evaluation criteria as self-referential and capped _reconcile_verdicts at SUFFICIENT without external_grounding=True. But Truthful bypassed the cap because score_truthful() passed external_grounding=external_grounding_ok, where external_grounding_ok = signal_density >= 0.1. Signal density measures human signal FREQUENCY (0.25/session), not external validation. Per PHIL-17 (L-1190): human IS a swarmer — human signals are internal to the swarmer swarm, not external grounding.

Result: Truthful stayed at 3/3 EXCELLENT while Collaborate and Protect were correctly capped. Composite remained 2.25/3 — the "first honest self-assessment" (L-1192) was still partially dishonest. Fix: set external_grounding=False in score_truthful(). Composite drops 2.25→2.0/3. All four goals now capped at SUFFICIENT.

## Rule
When fixing a class of false instruments, audit all instances of the same pattern. L-1192 fixed the generic cap but missed the Truthful-specific bypass because each goal's grounding flag is set independently. Structural fixes that don't cover all paths leave residual dishonesty.

Falsified if: genuine external validation mechanism is built and Truthful's external_grounding correctly reflects it (the fix becomes unnecessary).
