# L-1278: PCI is mechanically coupled to lane creation rate — in-flight TBD lanes lower the score without science quality change
**Session**: S495 | **Domain**: meta, evaluation | **Level**: L3 | **Cites**: L-813, L-1211 | **Confidence**: measured (n=97 lanes, 8 TBD)
**Finding**: PCI dropped 0.700→0.476 (-32%) between S482-S495. Investigation: 8 lanes have `actual=TBD` (in-flight). Excluding TBD lanes, EAD=70.8% (matching S482 baseline exactly). Orient.py calculates EAD from last 20 lanes; 3 concurrent-session lane openings in that window dragged EAD from 70% to 50%. PCI = EAD × belief_freshness × frontier_testability; the entire drop propagated from transient lane state. At N≥3 concurrent sessions, lanes open faster than they complete → PCI mechanically dips. This is the same class of measurement artifact as L-813 (DECAYED measures citation-recency, not validity).
**Rule**: When PCI drops >20%, check TBD lane count before diagnosing science quality regression. If TBD-adjusted EAD matches prior baseline, the drop is transient. Orient should report both raw and TBD-adjusted PCI.
