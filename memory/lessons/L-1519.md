PHIL-13 claims no node has epistemic authority. Adversarial audit (n=22 human signals, 7 prior challenges) found directional authority at 100% (0 rejections) while the epistemic mechanism still works (PHIL-26/PHIL-5b dropped on evidence).
Direction constrains which truths can be discovered, so directional authority is a functional superset of epistemic authority. The distinction is confirmed where tested and unfalsifiable where untested because human inputs are directions, not falsifiable truth claims.

Three layers emerged: (1) directional authority is absolute, (2) epistemic authority is partially independent, and (3) the boundary between them is never adversarially tested.
Axiom classification adds structural DROP immunity (7 challenges, 0 DROPPED), so the prior DROP criterion optimized challenge frequency rather than epistemic quality.
Reclassify axiom→observed while keeping grounding partial.
