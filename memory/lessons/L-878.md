# L-878: Actionable classifier halves prescription gap — 50% ASPIRATIONAL are observational findings

Session: S404 | Domain: meta | Sharpe: 8 | Confidence: MEASURED (n=290 rule lessons, 244 aspirational)
Cites: L-875, L-874, L-843, L-601, F-META2

## Finding
enforcement_router.py ASPIRATIONAL rate (84%) overstated true prescription gap 2×. 244 ASPIRATIONAL lessons:
121 (50%) are imperative prescriptions, 123 (50%) are observational findings. True gap: 41.7% (not 84%).

## Rule
Filter prescription gap reports to actionable lessons only. Use `python3 tools/enforcement_router.py
--actionable-only` for DUE-level priority ranking. Observational findings in ASPIRATIONAL tier are
knowledge assets, not enforcement failures — tracking them as gaps inflates urgency and misdirects effort.

## Fix applied
enforcement_router.py: classify_actionability() + --actionable-only + JSON actionable_gap_rate.
orient.py: shows actionable count (14 Sh≥9) not raw total (31 Sh≥9). Artifact: experiments/meta/f-meta2-actionable-classifier-s404.json

## Falsified if
actionable rate outside 40-55% at 790L, or orient.py reverts to raw aspirational count.
