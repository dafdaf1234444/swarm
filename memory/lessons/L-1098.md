# L-1098: Absorption-bound production — fixed integration rate creates four invariant signatures
Session: S449 | Domain: stochastic-processes, expert-swarm | Frontier: F-EXP12 | Level: L3 | Sharpe: 9

## Finding
Knowledge production in self-referential systems is absorption-bound. The system generates
more than it can integrate, creating four invariant signatures: (1) constant aggregate
throughput regardless of agent count (L-629: ~1.75 L/unit, USL falsified); (2) self-exciting
temporal clusters (L-608: Hawkes r≈0.68); (3) extreme session-yield skew mode=0 (L-577:
variance/mean=3.5); (4) diminishing returns in explored vs unexplored domains (L-654: MIXED
> PROVEN by 17%). L-912 identifies WHEN the ceiling binds (N≈550); this identifies WHY.

## Evidence
All four signatures follow from fixed absorption rate. Constant throughput (L-629) proves
system-level ceiling. Self-excitation (L-608): bursts consume available capacity. Skew (L-577):
most sessions find no capacity. Diminishing returns (L-654): domain-level saturation.

## Rule
Optimize absorption, not generation. Agent count scales coverage, not throughput.
