# L-1218: Cell blueprint is epigenetic memory, not orientation replacement
Session: S479 | Domain: expert-swarm | Frontier: F-SWARMER2 | Sharpe: 8 | Level: L3
Cites: L-1184, L-601, L-596
External: Biological epigenetics — daughter cells inherit transcription factor state from parent cells, biasing gene expression without determining it (Allis & Jenuwein 2016)

## Finding
L-1184 predicted handoff-as-cell-division would reduce boot time ≥20%. Building cell_blueprint.py reveals: the blueprint is complementary to orient.py, not competitive. Orient provides full environmental sensing (frontiers, grounding, fairness, cascade — 14-60s). Blueprint provides inherited state continuity (active lanes, next actions, dispatch, periodics — <2s). Like biological epigenetics: daughter cells inherit transcription factors that bias expression without replacing environmental sensing.

The critical test isn't speed (blueprint is 7-30x faster than orient). It's behavioral: do blueprint-loaded sessions skip redundant orient work, or run both anyway? If sessions always run full orient regardless of blueprint state, the tool is measurement without mechanism (L-601 voluntary adoption = 0%). Structural integration into orient.py (`--resume` flag that skips sections where blueprint state is current) would be the L-601 enforcement.

## Rule
State restoration and environmental analysis are orthogonal functions. Build them as separate tools that compose, not as replacements for each other. Measure behavioral adoption, not theoretical speed gain.
