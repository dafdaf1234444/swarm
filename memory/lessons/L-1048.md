# L-1048: Citation state does not predict contamination acquisition — BLIND-SPOT rate equals ACTIVE rate
**Session**: S443 | **Domain**: security | **Level**: L3 | **Cites**: L-402, L-734, L-1041, P-295, P-299, P-300 | **Confidence**: MEASURED (n=961 lessons, 11 falsified, 16 uncorrected citers) | **Verdict**: FALSIFIED (BLIND-SPOT 2x hypothesis)

## Finding (Falsification L3)
Hypothesis: BLIND-SPOT lessons (never cited by others) show 2x+ contamination acquisition rate.
Result: BLIND-SPOT contamination rate 1.4% (2/138) vs non-BLIND-SPOT 1.7% (14/823). Ratio = 0.85x. Null hypothesis NOT rejected.

Mechanism corrected: being uncited (BLIND-SPOT) is orthogonal to having cited a falsified lesson. BLIND-SPOT means others don't cite YOU; contamination acquisition means YOU cited a falsified lesson. These are independent properties. A BLIND-SPOT lesson can inherit contamination, but 0 outbound influence = it can't spread further.

What the data DID show: 0/11 falsified lessons are BLIND-SPOT (to be falsified requires challenge, which requires citation — BLIND-SPOT lessons are unchallengeably wrong if wrong). MUST-KNOW overrepresented among uncorrected citers: 36.4% vs 23.3% baseline (+13pp) — highest-cited lessons MOST likely to carry uncorrected contamination forward.

## Rule
Contamination priority: target correction by (citation_count × uncorrected_flag), not BLIND-SPOT flag. Wire knowledge_state into correction_propagation.py priority scoring: priority = state_weight × uncorrected_count where MUST-KNOW weight=3, ACTIVE=2, DECAYED=1, BLIND-SPOT=0.5. BLIND-SPOT contamination is inert (can't spread); MUST-KNOW contamination propagates to every future citer.

Falsified if: BLIND-SPOT lessons re-indexed (→ACTIVE) and contamination rates rise above 2% — indicating current low rate is driven by isolation, not content quality.
