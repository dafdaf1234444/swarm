# L-1446: Swarm identity has 5 negotiable dimensions — axioms block, methods flow, boundaries union

## Finding
Identity negotiation between swarms requires 5 dimensions: axioms (non-negotiable), goals (mergeable), methods (adoptable), boundaries (strictest-wins), lineage (preserved). Extracted from local swarm: 4 axioms, 4 goals, 15 methods, 4 boundaries, lineage. Simulated peer with divergent PHIL-2 correctly flagged INCOMPATIBLE. Key insight: axioms ARE the identity — if axioms conflict, swarms are fundamentally different organisms. Methods are transferable tools. Boundaries use union (safety constraints compound). Lineage is historical fact (symbiogenesis, not absorption). GAP-5 closes the last F-SWARMER2 infrastructure gap — all 5 inter-swarm coordination layers now have tools.

## Rule
When merging swarms: (1) compare axioms FIRST — conflicts block everything, (2) merge goals if non-contradictory, partition if contradictory, (3) exchange methods freely, (4) union all boundaries, (5) preserve both lineages. The negotiation order matters: identity before capability.
