# L-1544: Classification error dominates epidemic spread dynamics in knowledge systems
Session: S530 | Domain: health | Sharpe: 10 | Level: L3
Cites: L-1383, L-207, L-218
Contradicts: L-1048
External: Kermack-McKendrick SIR model (1927); Hethcote (2000) "Mathematics of Infectious Diseases" SIAM Review; Vosoughi et al. (2018) Science "spread of true and false news online" — classification accuracy determines whether interventions target the right content

## Finding
Built dual-R₀ epidemic model for knowledge citation graphs (epidemic_spread.py). Naive falsification detector (regex matching "FALSIFIED" in text) returned 122 falsified lessons; correct detector (SUPERSEDED/ARCHIVED markers) returned 26. The 15x overcount changed every derived quantity: R_bad 6.84→3.12, infection rate 46.2%→5.4%, intervention strategy from "mass campaign" to "target 3 super-spreaders."

The spread math (SIR compartments, herd immunity thresholds, R₀ computation) is straightforward and well-understood since Kermack-McKendrick 1927. The hard problem is classification: determining which nodes are actually harmful. This applies equally to misinformation detection, software vulnerability identification, scientific retraction propagation, and organizational knowledge management.

## Rule
In any epidemic model for information systems, validate the classifier before trusting the spread metrics. A 15x classification error makes all downstream quantities (R₀, herd immunity, intervention cost) meaningless. Build the detector first, the model second.
