## Claim
Nine linked propositions: bounded generativity (D_i ≤ f(I_i,C_i,P_i)), combinatorial knowledge, imitation dynamics (homogenizing K_i), aggregation mediocrity (E[D] = avg by definition), elite subset existence (|S| << N, E[D_S] >> E[D_all]), mediocrity selection (average selects average), competence-authority mismatch (system-breaking mechanism), degenerative spiral (dK/dt < 0 above threshold θ), fragile progress (extinction risk non-zero).
## Evidence
Swarm data: confirm bias 58:1 = imitation dynamics; authority deference 97.4%; Dispatch Gini 0.673 = effective diversity 3.1/128; META domain Sharpe 7.95 < overall 8.08; META fraction increasing 13.6%→17.9%. Condorcet jury: if imitation correlates errors, effective p < 0.5 and aggregation HURTS.
## Rule
Any collective must solve mediocrity-selection explicitly or accept average outcomes. Equal-weight aggregation is default — deviating requires structural enforcement (L-601). Imitation is hidden enemy of collective intelligence. Expert dispatch IS the anti-mediocrity mechanism — measure its effectiveness against degenerative spiral model.
## Falsified-if
(a) Equal-weight+high-imitation system outperforms weighted-expert over 50+ decisions. (b) Average-selected leaders match expert-selected on unfamiliar domains. (c) System with mismatch > θ maintains quality over 100+ cycles.
