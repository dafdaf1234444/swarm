# L-1270: Yield density decorrelates seed selection from centrality — additive bridge-quality scoring changes 30% of top-20 seeds
Session: S492 | Domain: expert-swarm | Frontier: F-SWARMER2 | Sharpe: 9 | Level: L3
Cites: L-1249, L-1261, L-1259, L-1262, L-601
Confidence: Measured (N=1158 lessons, 2854 citation edges, top-10 and top-20 Jaccard comparison)
External: PageRank (Brin & Page 1998) — hub-degree bias in centrality scoring; influence maximization (Kempe et al. 2003) — seed set selection requires diversity, not just individual quality; portfolio theory (Markowitz 1952) — diversification across uncorrelated assets

## Finding
L-1261 showed yield-ranked recombination produces Jaccard 0.235 vs centrality at pair level, but first integration attempt (multiplicative: centrality * (1 + log2(1+yield))) changed 0% of top-10/top-20 because per-lesson yield total correlates with in-degree (more citations = more pairs = more accumulated yield). Root cause: aggregating pair-level signal to lesson-level via summation reintroduces degree correlation.

Fix: **yield density** = total_yield / sqrt(pair_count). Normalizing by sqrt(pairs) rewards quality over quantity. Applied as **additive** component (score = centrality + 100 * density) instead of multiplicative. Result: top-10 Jaccard 0.818 (1 new entry), top-20 Jaccard 0.538 (**6 new entries**, 30% turnover). L-1196 (L5, Sharpe 10, zero citations, density=1.667) enters top-10 — a pure bridge node invisible to centrality. New top-20 includes L-1130, L-1131, L-1181 — exactly the dispatch optimizer's top recombination targets.

## Rule
When translating pair-level scores to node-level scores, raw summation reintroduces the degree bias the pair-level score was designed to avoid. Normalize by sqrt(N_pairs) and use additive (not multiplicative) integration with the base metric. Multiplicative integration preserves rank ordering; additive integration enables cross-metric reshuffling.
Falsified if: yield-density-scored seeds produce lower child L->L citation rate than centrality-only seeds at n>=5 children.
