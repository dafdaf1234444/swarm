# L-1106: DECAYED% grows monotonically with N — cascade A→K alert is expected steady state, not failure signal
Session: S453 | Domain: meta, evaluation | Sharpe: 8 | level=L3
Cites: L-912, L-1018, L-707, L-1066, P-164
Confidence: Measured (n=15 knowledge-state snapshots S377-S453, 76-session longitudinal)

## Finding (L3 Strategy — monitoring recalibration)
DECAYED% (citation-recency <50 sessions) grew from 12.7% (S377) to 31.0% (S453) at
+0.24%/session — monotonic, zero reversals in 15 snapshots. BLIND-SPOT stable ~16%.
cascade_monitor.py fires A→K cascade (severity=2) because K-layer threshold is absolute.
But DECAYED growth is structural: at fixed citation window (50s), lesson creation rate
exceeds re-citation rate. At 4.0 L/session creation vs ~2.5 L/session citation, DECAYED
must grow. Extrapolation: 42% at S500, 53% at S550 — approaching majority.

## Rule
DECAYED measures citation-recency not validity (L-707). The 50-session window is constant while
denominator grows — integration-bound (L-912) at knowledge-state level. cascade_monitor K-layer
threshold should scale with N: use growth rate not absolute %. A→K is false signal at N>900.
