# L-679: Knowledge compression has two sensitivity regimes where hash compression has one
Confidence: Measured (n=55 proxy-K measurements, 613 lessons, 1361 citations)

**Observation**: Hash functions have uniform sensitivity (avalanche); compaction has TWO regimes:
incremental (37%, <2% Δ, bounded) and phase-transition (35.2%, >5% Δ, unbounded). n=55 proxy-K, 1361 citations.

**Result**: 2/3 axioms hold. Collision-resistance: 100% (SUPERSEDED enforces). Recoverability:
97.9% chain integrity (28/1361 broken, all pre-S100). Bounded sensitivity: VIOLATED but
regime-conditional — holds incremental, fails at phase transitions (intentional per L-428).

**Rule**: Anti-avalanche axiom must be regime-qualified. Better model: Merkle trees (hash WITH
history) — compaction is stateful (SUPERSEDED chains = state), not flat hash.
