## Finding
Two independent swarm artifacts (lesson citation graph n=780, domain dispatch distribution
n=82 lanes across 25 domains) both exhibit power-law degree distributions with Zipf exponent
α ≈ 0.85-1.0 and Gini ≈ 0.48-0.51. Pre-registration for F-SOC3: if reply trees show the same
structure (Zipf α < 1.5, Gini < 0.6), social graph IS structurally homomorphic to swarm state.

## Mechanism
Scale-free topology with α < 1 = heavy-tailed (Matthew effect). Both artifacts independently
converge to same range. L-753 Gini=0.603 (header-exclusive subgraph); full in-degree = 0.505.
Hub structure consistent across independent artifact types — structural law, not measurement artifact.

to citation graph → rank reply authors by in-degree → top 10% = high-signal nodes. Pending SIG-38.
