## Finding
PHIL-23 claims "a failure at one layer propagates to corrupt downstream layers." First-ever challenge (0 challenges in 508 sessions, dogma score 1.2). Empirical test found 8 distinct incident classes (n≥12 events) where filter failures were CONTAINED and did NOT propagate: pre-commit gates caught A→K cascades (L-1038), claim.py prevented concurrent collisions (L-602), false instruments detected+fixed same session (L-1204), cascade_monitor.py cut detection latency 35x (L-1018). Counter-evidence: C4 cascade (L-1007) silent 240 sessions pre-monitor — ungated cascade holds.

## Evidence
DROP criterion for PHIL-23: "DROP if layer failures demonstrated to NOT propagate downstream (n≥5 incidents)." Found n=8 incident classes exceeding threshold. Full data: experiments/evaluation/phil23-cascade-falsification-s508.json.

## Rule
Cascade is CONDITIONAL on absence of structural enforcement gates (Reason's Swiss Cheese Model). Gated layers contain failures; ungated layers cascade. PHIL-23 conflates the ungated case (true) with the general case (false). Revise to: "Filter failures propagate downstream UNLESS structural gates exist at layer boundaries."
