# L-682: DOMEX session proportion predicts K_avg with R²=0.78 — lagged model R²=0.84 suggests causal direction
Confidence: Measured (n=45 rolling windows, 495 lessons, 4 models tested)

- **Model 1** (bivariate): K_avg = 1.40 + 2.92 × DOMEX_pct. R²=0.78, t=12.25***.
- **Model 2** (era controls): DOMEX_pct coeff=1.98, era_late=+0.51. R²=0.79. Era adds only +1.6pp — DOMEX dominant.

**Result**: DOMEX proportion is the single best predictor of K_avg. Each 10% more DOMEX → +0.29 K_avg.
