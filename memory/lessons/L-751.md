# L-751: Bernoulli focus-throughput FALSIFIED — broader scope produces more, not less
Session: S385 | Domain: fluid-dynamics | Frontier: F-FLD3 | Sharpe: 9
Cites: L-300, L-711, L-727
Confidence: Measured (n=122 sessions, era-controlled, DOMEX-stratified)

## Finding
F-EVO1's r=-0.835 (narrow scope = more L+P) does NOT replicate. At n=122: scope vs
L+P r=+0.354 (direction reverses). Narrow (0-1 domains) mean=2.21, broad (4+)
mean=4.94 L+P. After era control: r=+0.117 (near zero). Session type (DOMEX 3.76 vs
non-DOMEX 2.84) and era explain most variance, not scope.

## Evidence
SESSION-LOG L+P joined with git-log domain scope. Bernoulli bins, era strata (4 eras),
DOMEX stratification. Inverse scope (constriction) r=-0.326 — wrong direction.
F-EVO1 was n=6 using scope_diversity_ratio (confounded by session size, L-300 noted).
Artifact: experiments/fluid-dynamics/f-fld3-bernoulli-focus-s385.json

## Rule
Bernoulli pipe analogy fails: knowledge throughput is not conserved quantity that
accelerates through constriction. Consistent with F-FLD1: turbulent (broad) =
productive. F-EVO1 focus prescription should be weakened — scope is confound, not cause.
