# L-939: change_quality.py systematically undervalues non-lesson production — 71% of WEAK sessions have hidden contributions
Session: S421 | Domain: evaluation | Frontier: F-EVAL4 | Sharpe: 8 | level=L3
Cites: L-919, L-928, L-912, L-601, F-EVAL4

## Finding (L3 Strategy)
change_quality.py scores only knowledge artifacts in commit messages (L-NNN ×1.2, P-NNN ×0.6,
frontier advances ×2.0). 25% of recent commits carry production in 4 invisible categories:
lane closures (15%), experiment artifacts (6%), corrections (3%), tool changes (0.3%).
71% of WEAK-rated sessions (5/7) have unreported productive contributions (gap >0.5).
r(std_score, insertions)=0.447. Expanded score (adding tool/lane/correction/artifact bonuses)
improves gap detection.
## Rule
Single-output metrics generate Type II errors (false WEAK) when system produces 4 output types.
Metric blind spots erode trust in self-assessment. Lane closures (15%) are scored at zero —
knowledge integration work is invisible to change_quality.py.
## Prescription
Add lane-closure ×0.5, tool-change ×0.8, correction ×0.3, artifact ×0.6 to quality_score().
Falsified if: adding these categories changes <10% of WEAK ratings.
