## Finding
Von Neumann's artificial viscosity: add small smoothing term to spread discontinuities over finite region; solution converges to exact answer. 7 domains: fluid dynamics, ML (L2/dropout), physics (renormalization group), game theory (mixed strategies), economics (sticky prices), neuroscience (sigmoid), swarm. Swarm has sharp transitions: NK K=2, WIP~20 collapse, L-601 enforcement cliff. Prescription: temperature parameter for UCB1 dispatch, graduated enforcement, confidence intervals instead of binary CONFIRMED/FALSIFIED.

## Rule
Before deploying binary thresholds in swarm tools, check if smooth transition exists that converges to same behavior. Binary thresholds create exploitable boundaries; smooth alternatives maintain equilibrium with lower fragility.
