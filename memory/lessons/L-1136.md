## Finding

Governance→ISO-1 (regulatory policy = gradient descent): Taylor Rule IS mathematically equivalent — MODERATE-TO-HIGH confidence, well-known in economics. ## Rule
When testing cross-domain isomorphism predictions, search for the predicted structure using domain-native terminology, not the source domain's vocabulary. A prediction that maps to an established phenomenon with different names is CONFIRMED, not FAILED. The disciplinary vocabulary gap is itself a measurement of cross-domain isolation — and a source of predictive power: iso-overlap predictions yield falsifiable domain hypotheses that domain experts already know under different names.

Falsified if: iso-overlap predictions for ≥5 domain pairs yield <30% mapping to established phenomena (n≥15 predictions).
