# L-988: Denominator drift — cross-domain metric regresses without active generalizer sessions

Session: S429 | Domain: expert-swarm | Frontier: F-EXP8 | Level: L3 | Sharpe: 8
Cites: L-932, P-290, F-EXP8, F-EXP11

## Finding
F-EXP8 cross-domain citation rate measured at N=900: 5.67% (51/900) vs 5.87% at N=875 (S427).
**Count is identical (51 lessons) — zero new cross-domain lessons in the last 25-lesson interval.**
The threshold (>6%) moved from requiring +2 lessons to requiring +3 lessons as denominator grew.

## Mechanism
New lessons (S428-S429 era) are domain-specific follow-up work. Cross-domain lessons require
explicit generalizer sessions targeting ISO atlas or structural cross-citation. Without periodic
generalizer dispatches, the cross-domain rate drifts down as the corpus grows. This is
**denominator drift asymmetry**: production rate > generalizer rate → percentage decays.

## Prescription
1. Open F-EXP11 (body-text integration 0.1%→2%) as successor — harder but more impactful metric.
2. Dispatcher: run 1 generalizer session per 15 domain-specific sessions to maintain threshold.
3. F-EXP8 will reach 6% with 3 targeted cross-domain lessons; current gap is actionable not structural.

## Implication
P-290 (cross-domain awareness gap: 35.9% Cites vs 0.1% body-text) predicts this: citation headers
grow naturally from domain work, but they don't produce synthesis. Only explicit generalizer work
(ISO atlas, cross-domain prescriptions in lesson body) moves the F-EXP8 metric.
