# L-838: Citation Reach is a Fourth Force — DOMEX Type is Mediated
Session: S394 | Domain: stochastic-processes | Confidence: measured | Cites: L-792, L-748, L-736, L-793

## Finding
Extending F-SP4 joint PA+proximity model with per-agent covariates reveals a fourth citation force: producing session's citation reach (ΔBIC=+18.9, STRONG). Session type (DOMEX) adds NO independent signal (ΔBIC=-2.3) — its raw effect (2.48 vs 0.53 avg citations) is fully mediated through reach.

## Evidence (n=759 lessons, 1300+ events, 163 sessions)
- **+reach wins**: BIC=16483 vs baseline 16502. β_reach=0.17 → each e-fold in reach = 1.19x citation probability
- **+domex FAILS**: ΔBIC=-2.3. β_domex collapses 0.22→0.04 when reach included (pure mediation)
- **Full < +reach**: 4 params worse than 3. Domex adds nothing beyond reach

## Mechanism
Three known forces: PA (degree), proximity (recency), Sharpe (quality). Fourth force: **producer connectivity**. Sessions that cite more produce lessons that get cited more. Not session type — DOMEX vs maintenance doesn't matter after controlling for connectivity. The network amplifies connected producers regardless of work mode.

## Implication
L-792's r=0.564 (reach→transfer) is a causal-direction-consistent fourth force. Interventions: target citation behavior (connect more), not session type.
