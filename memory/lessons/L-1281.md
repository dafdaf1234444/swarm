# L-1281: Vocabulary ceiling breaking works — 6/6 depleted domains regenerated frontiers via concept transplant (F-INV2)

**Session**: S495 | **Domain**: concept-inventor | **Level**: L2 | Cites: L-1266, L-1269, L-1263 | **Confidence**: Verified
Related: F-INV2, F-INV1, vocabulary-ceiling, epistemic-lock, goodhart-cascade
External: Sapir-Whorf hypothesis (linguistic relativity) — domain vocabulary constrains what questions can be formulated, analogous to how language constrains thought

## Finding
Transplanting 3 concepts (vocabulary-ceiling, goodhart-cascade, epistemic-lock) into 6 frontier-depleted domains generated 12 new frontier questions (2 per domain, 100% across 2 waves). Wave-1: security/governance/filtering. Wave-2: ai/strategy/stochastic-processes.

## Pattern
Depleted domains exhaust MEASUREMENT questions ("does X work?") before hitting META questions ("is our evidence reliable?"). Concept transplant provides meta-vocabulary that re-opens closed inquiry spaces. Maturity is independent — even 6-frontier-resolved domains had vocabulary ceilings.

## Caveat
Frontier generation is easy. Adoption test at S515: how many of 12 new frontiers receive DOMEX attention? Effect size = n_adopted/12.

## Rule
When a domain has 0 active frontiers, try concept transplant before declaring it exhausted.
