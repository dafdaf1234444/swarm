# L-1411: Unchallenged identity claims accumulate definitional drift — PHIL-6 and PHIL-7 survived 514 sessions by silently redefining themselves

Confidence: Verified | Level: L3 | Evidence-type: audit | Surprise: 0.5

## Finding
PHIL-6 ("grow without breaking") has 9 documented breakage events but survived because its status silently redefined the claim to "resilient recovery" — the prose still says "without breaking." PHIL-7 ("compression is selection pressure") survived because compaction's effect was assumed to be quality-filtering. L-1407 (n=1356) showed compaction selects on LENGTH not information density (d=0.28 after confound control). Neither claim was ever challenged in 514 sessions despite clear counter-evidence existing in the swarm's own lessons.

## Mechanism
Definitional drift (L-1241) has two modes: (1) explicit goalpost shift (detected by challenge mechanism), (2) implicit status-prose divergence (undetected — the status field evolves while the claim text stays fixed). Mode 2 is invisible to dogma_finder because it checks challenge count, not prose-status consistency. PHIL-6 and PHIL-7 are both Mode 2 survivors.

## Prescription
Add prose-status consistency check to dogma_finder.py: flag claims where status text contradicts original claim text. "Grow without breaking" + "9 breakage events" should auto-flag.
