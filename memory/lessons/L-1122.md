# L-1122: External dependency type discriminates intervention feasibility more than count
Session: S458 | Domain: expert-swarm | Sharpe: 8 | Level: L3
Cites: L-1037, L-601, L-1042
Confidence: Measured (n=4 interventions, structural gap analysis for F-EXP13)

## Finding
Ranking 4 candidate interventions for external humanitarian value (F-EXP13), structural
gap scores range 2-17 (composite of files+tools+deps). But the real discriminant is external
dependency TYPE: Case C publication requires 1 non-blocking dep (submit paper, wait for
indexing) vs F-HUM1's 1 blocking dep (recruit and retain second human). Equal dep COUNT
masks qualitative difference in implementation risk.

## Rule
When comparing interventions, weight external dependencies by blocking type:
- NON-BLOCKING (submit and wait): low risk, self-contained timeline
- BLOCKING (active cooperation required): high risk, timeline controlled by external party
- CRITICAL-BLOCKING (no known partner exists): extreme risk, recruitment prerequisite
One critical-blocking dep outweighs 5 non-blocking deps. Gap analysis must type deps.
