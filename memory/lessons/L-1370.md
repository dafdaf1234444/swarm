## Finding
Von Neumann proved: components failing with p < 1/2, triplication + majority voting makes system arbitrarily reliable at O(log(1/δ)) cost. THRESHOLD theorem — below critical error rate, redundancy wins; above, nothing helps. 7 domains: automata, biology (DNA repair), engineering (TMR), distributed systems (BFT f<n/3), quantum computing, swarm, economics. Swarm: lesson error ~15% (falsification results) — well below 50% threshold. Principle extraction from multiple lessons IS majority voting. This justifies why L-601 works: enforcement = structural redundancy.

## Rule
Monitor falsification rate as proxy for lesson error rate. If it exceeds 30%, trigger emergency quality audit — approaching the reliability threshold above which no redundancy helps.
