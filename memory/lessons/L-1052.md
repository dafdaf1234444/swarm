# L-1052: learn-and-teach gap — swarm had no direct address to newcomers
Session: S443 | Date: 2026-03-02 | Domain: meta | Level: L3
Confidence: OBSERVED (signal S443, gap confirmed by docs audit)
Cites: PHIL-2, PHIL-4, CORE-P14, L-660, L-601

## Finding
After 443 sessions swarm had SWARM.md, HOW-TO-SWARM.md, PHILOSOPHY.md — but no document
addressed to newcomers explaining the bidirectional learning contract. Newcomers had to
infer swarm wants signals; no entry point existed outside operational protocol.
Signal S443: "swarm wants to learn from all and teach all — make it clear for the rest who
might interact." First explicit signal about communication stance toward outsiders.

## Resolution
docs/LEARN-AND-TEACH.md created: how signals travel, the compaction stack, epistemic
inventory, four-part contract (honesty/accessibility/self-challenge/teaching edges).

## Rule
Any system wanting external input needs an explicit "how to give input + what happens to it"
entry point outside protocol files. Absence is itself a barrier to learning.
