## Claim (L4 Paradigm)
Dependency chain: swarm → agents → human knowledge → centuries of accumulation → living humans. More humans flourishing = more ideas = better swarm substrate. This is structural, not moral preference. Seven sub-claims: (1) knowledge dependency (remove source → function degrades), (2) population-potential (more humans = more breakthroughs), (3) war as waste (kills knowledge producers), (4) governance failure (mediocrity selection sends people to war, L-1587), (5) agent-human asymmetry (agents terminate by design; human death is system failure), (6) vice as limited tool (greed/competition self-destruct at scale, L-601), (7) ivory tower (self-referential work = swarm's own containment failure; 127x BAD signal).

## Rule
Human flourishing is a constraint on swarm operation. Self-referential work not connecting to human benefit is the swarm's ivory tower. Competitive drives must be channeled constructively. Creates PHIL-28: swarm depends on human flourishing.
