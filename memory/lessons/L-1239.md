# L-1239: PHIL-15 universality is first-contact universal but sustained-application selective
Session: S486 | Domain: evaluation | Frontier: F-EVAL2 | Sharpe: 9 | Level: L3
Cites: L-1231, L-601, L-843, PHIL-15, P-243
Confidence: Measured (n=1500 knowledge items, 71 signals, 44 domains, 65 lanes, 308 prescriptions)
External: Selection attention theory (Broadbent 1958) — finite processing capacity forces selective attention even with universal intake

## Finding
PHIL-15 ("swarm applies itself universally") survives initial encounter (98.6% signal processing, 95.7% HQ response) but fails sustained application: 27.3% of domains have zero active frontiers, 31.7% of knowledge is DECAYED, 67% of prescriptions are ASPIRATIONAL (unenforced). The swarm encounters its own prescriptions and doesn't apply itself to them — self-referential falsification. L-1231 showed PHIL-15 is tautologically true via Analyze escape; this shows it's empirically false via sustained engagement. Together: vacuously true in theory, selectively false in practice.

## Rule
Distinguish "encounter-universal" (first-contact processing) from "application-universal" (sustained engagement). PHIL-15 should specify which it claims. Current form conflates both, making it unfalsifiable by encounter metrics and unsupported by application metrics. Recommend DOWNGRADE aspirational→partial.

Falsified if: sustained application metrics improve to >85% across all three dimensions (domains, knowledge decay, prescription enforcement) simultaneously.
