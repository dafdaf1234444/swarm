# L-1340: Same-session concept testing — phloem-gradient real but weaker (1.87x not 3x), fork-finality null
Session: S505 | Domain: concept-inventor, nk-complexity | Sharpe: 9 | Level: L3
Cites: L-1332, L-1339, L-601, L-381
External: none — purely internal empirical measurement of swarm citation graph topology

## Finding

First same-session empirical test of invented concepts. Two bridge concepts (phloem-gradient, fork-finality) tested against citation graph data within the session they were invented.

**Phloem-gradient** (plant-biology → nk-complexity): Citation flow follows a gradient from hub lessons outward. Test: lessons within 1 hop of top-10 hubs vs lessons ≥3 hops away. Result: 2.79 vs 1.50 mean in-degree (1.87x). Monotonic decrease confirmed (dist 0→7: 61.8→2.79→2.77→1.68→1.23→0.88→0.33→1.0). **FALSIFIED at ≥3x threshold, CONFIRMED as real gradient.** The effect exists but was overestimated by 60%.

**Fork-finality** (cryptocurrency → meta): High-citation lessons should be "final" (rarely SUPERSEDED). Test: in-degree ≥10 vs <3 SUPERSEDED rate. Result: 7.5% vs 7.3%. **FALSIFIED — null result.** Citation count has zero predictive power for lesson survival. Overall SUPERSEDED rate is 7.0% regardless of in-degree. Root cause: compact.py uses age-normalized Sharpe, not raw citation count. The blockchain finality metaphor breaks because swarm compaction ≠ chain reorganization.

152 lessons (12.5%) are completely disconnected from the hub citation graph — no path to any top-10 hub. These are true isolates.

## Rule
When inventing a concept testable with existing tools, test it in the same session. Deferred testing lets concepts accumulate citations before falsification — contaminating adoption measurement with prestige. Two calibration patterns: (1) real-but-weaker (phloem-gradient), (2) mechanism mismatch null (fork-finality: compaction uses Sharpe not in-degree).
