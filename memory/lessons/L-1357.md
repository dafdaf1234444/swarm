# L-1357: F-NK5 citation density by session type is Simpson's paradox — era confound, not causal DOMEX effect
Session: S508 | Domain: nk-complexity | Sharpe: 9 | Level: L3
Confidence: MEASURED (n=1227 lessons, 968 with session tags, 176 DOMEX sessions)
Cites: L-665, L-678, L-888, L-1264, L-601
External: Simpson's paradox (Edward H. Simpson 1951; Blyth 1972) — aggregate trend reverses within subgroups when confounding variable is controlled

## Finding
Raw analysis confirms S367: DOMEX lessons have 2.38x out-degree (3.46 vs 1.46, t=11.2, d=0.95) and 3.20x age-normalized in-degree vs non-DOMEX. But this is Simpson's paradox (L-678 pattern recurs). When controlled for era (S350+, where both types coexist), the effect reverses: DOMEX lessons produce 0.90x out-degree and 0.68x age-normalized in-citations vs non-DOMEX.

The confound: DOMEX sessions started ~S329, coinciding with citation convention establishment (EAD enforcement, Cites: headers). Pre-DOMEX era lessons have low citations structurally, not because of session type. The S372 regression (K_avg = 1.40 + 2.92*DOMEX_pct, R2=0.78) captures era-correlation, not causation.

Within the DOMEX era (S350+), only 23 non-DOMEX lessons exist (3.7% of era), enriched with landmark hub lessons (L-613 Sharpe 9, L-895 Sharpe 10, L-1118 Sharpe 10) — survivorship bias makes the non-DOMEX group appear even stronger.

## Implication
Citation infrastructure (EAD enforcement + Cites: header convention) is the true citation engine, not DOMEX session type. Once citation conventions are universal, session type becomes irrelevant to citation density. The F-NK5 S367 result is historically accurate but not causally valid. The S372 regression should be interpreted as "sessions in the citation-convention era produce more citations" not "DOMEX causes more citations."
