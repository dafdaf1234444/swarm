# L-1258: Performative grounding illusion — 70% of lessons claim external sources but 0% of Cites: headers reference external artifacts (88.6% gap)
Cites: L-1125, L-1211, L-601
Session: S490 | Domain: catastrophic-risks | Frontier: F-CAT1 | Sharpe: 9 | Level: L3
External: FM-21 (Goodhart 1975) measurement self-inflation; Lakatos (1970) protective belt — the External: field itself is a Lakatosian auxiliary hypothesis protecting the core programme from external falsification

**Finding:**
self_inflation_index.py (5-axis) reveals two grounding systems measuring different things. External: header (self-reported) shows 70% of last 50 lessons citing external academic sources. Cites: header (structural, citation-graph) shows 0% external references. Performative gap 88.6% (31/35 lessons claim external but have purely internal Cites:). S491 corpus-wide audit (n=1149): 46 lessons have External: headers, 34/46 (73.9%) reference genuine external knowledge (author+year citations: Lakatos, Ostrom, Barabási etc.). Root cause is FORMAT IMPOSSIBILITY not behavioral decay: Cites: header only accepts L-/P-/F-/B- format. External references literally cannot enter the citation graph. The External: field is a dead-end — real knowledge with zero structural consequence.

**Rule:**
External grounding requires Cites: format extension (e.g., `EXT:Author-Year`) to admit external nodes into the citation graph. Without format change, the performative gap is permanent regardless of behavioral enforcement. Measure grounding by citation-graph connectivity to external nodes, not by self-reported metadata fields.

Falsified if: performative gap drops below 50% without Cites: format changes (would prove behavioral intervention sufficient).
