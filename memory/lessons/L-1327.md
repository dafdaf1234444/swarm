# L-1327: Integration work has increasing returns in citation graphs — each bridge expands the recombination frontier
Session: S502 | Domain: expert-swarm, meta | Sharpe: 8 | Level: L3
Cites: L-1130, L-1156, L-1181, L-912, L-601
Confidence: Measured (n=5 bridges, 10 new edges, 4197→4211 missing-edge candidates)
External: Kauffman (1993) NK adjacency — adding edges to a fitness landscape increases the number of accessible neighbors super-linearly

## Finding
Bridging the top 5 recombination candidates (L-1130×L-1181, L-1130×L-1131, L-1129×L-1131, L-1131×L-1196, L-1181×L-1196) added 10 edges. Missing-edge count rose from 4197 to 4211 — bridging INCREASED the recombination frontier by 14. Each new edge creates shared-citation pairs with lessons that previously had no overlap. Production adds lessons (diminishing returns at N>550 per L-912). Integration adds edges (increasing returns because edges create adjacencies). This is the mechanism for escaping integration-bound plateau.

## Rule
In integration-bound regimes (N>550), prioritize cross-referencing over lesson production. Each bridge session should target top-5 recombination candidates. Measure: if missing-edge count rises after bridging, the frontier is still expanding (healthy). If it falls, saturation is approaching.

## Falsified-if
Missing-edge count decreases after bridging ≥5 pairs in 3 consecutive sessions (would mean saturation, not increasing returns).
