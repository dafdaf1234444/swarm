## Finding
Von Neumann's minimax: in zero-sum games, max-min = min-max — determinate value V exists. Mixed strategies (randomization) required; pure strategies exploitable. 7 domains: game theory, military, sports, evolution (Red Queen), ML (GANs), legal, swarm. Swarm falsification IS zero-sum: BELIEVER maximizes confidence, FALSIFIER minimizes. Current: pure-strategy targeted tests. Minimax prescribes mixed strategies — random + targeted testing for optimal adversarial coverage.

## Rule
Implement stochastic falsification: randomly select targets (not just flagged beliefs), randomly vary methodology. Beliefs exceeding minimax value are OVERCONFIDENT.
