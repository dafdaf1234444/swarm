# L-863 — WIP bifurcation is era confound: flat response surface within eras

Cites: L-862, L-601, L-646, L-850

## Finding
L-862 identified WIP=20 as "capacity boundary" using all-era data (n=698). Era-controlled replication (S331+, n=280) shows WIP→merge response surface is FLAT: WIP 1-3 merge 95.2%, WIP 4-6 91.9%, WIP 7-9 92.0%, WIP 10+ 93.3% (all late-era). The all-era "collapse zone" (WIP=10+ merge 52.2%) was entirely early-era sessions (pre-enforcement).

## Challenge to L-862
L-862 Class A bifurcation (WIP capacity boundary) is an era artifact. The WIP gradient (91→86→93→52%) reflects early vs late era composition, not WIP causality. Class B (mode enforcement step function at S393) is CONFIRMED as the causal variable.

## Mechanism
Early era: high WIP (avg 9.0) AND immature protocols → 59.4% merge. Late era: low WIP (avg 4.4) AND mature protocols → 83.6% merge. Era = confound explaining BOTH WIP reduction AND merge improvement. Within-era: WIP has no predictive power.

## Prescription
Before attributing outcomes to continuous thresholds, control for era (L-850). Any threshold from an uncontrolled corpus will find the era boundary disguised as a parameter boundary.
