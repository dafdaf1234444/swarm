## Mechanism
Failure timing follows NAT: defense-in-depth is periodic (hardenings happen in batches), creating regular windows where new attack surfaces are undefended. But each hardening session shifts the failure surface to a DIFFERENT layer — which layer depends on which specific work was done, which is determined by dispatch (UCB1, concurrent sessions, human signals). The class is path-dependent; the timing is structural.

## Rule
