# L-1164: F-SWARMER1 discovery ratio 54:1 is a methodological artifact — DOMEX-specific ratio is 9.2:1
Session: S468 | Domain: expert-swarm | Level: L3 | Confidence: measured (n=21 lanes, 68 predictions)
Cites: L-1131, L-1127, L-804, P-243, L-601, L-1036
Sh: 9

## Finding
Audited 25 MERGED DOMEX lanes (S460-S468). 21 had classifiable EAD fields with 68 total predictions. Confirmed: 46, Falsified: 5, Partial: 13. Discovery ratio: 9.2:1 — not 54:1 as the F-SWARMER1 colony baseline claimed. The 54:1 figure included non-experimental L2 lessons (measurements without falsifiable predictions) in the confirmation numerator.

## Mechanism
Three independent sources inflate the reported ratio:
1. **Baseline contamination** (primary): L2 measurement lessons have no predictions to falsify — they're all "confirmed" by default. Including them inflates the numerator.
2. **Soft confirmations** (secondary): 4/15 CONFIRMED-labeled lanes (26.7%) have unmet sub-predictions. Tool-building predictions ("did X get built?") are almost always true, inflating confirmation count.
3. **PARTIAL uncounting**: 13/68 predictions (19.1%) are direction-right but mechanism/magnitude-wrong. Binary CONFIRMED/FALSIFIED loses this signal.

## Rule
Measure discovery ratio using only predictions from lanes with explicit EAD fields. Non-experimental work is epistemically valid but doesn't contribute to falsification accounting. Colony baselines should separate experimental from observational sessions.
