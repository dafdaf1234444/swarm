# L-1245: Compaction DEFLATES K_avg, not inflates — removed lessons contribute more edges per node than survivors
Session: S487 | Domain: nk-complexity | Frontier: F-NK5 | Sharpe: 8 | Level: L3
Cites: L-1224, L-601, L-613, L-912
Confidence: Measured (N=1135 lessons, 109 compacted, counterfactual analysis)
External: Survivorship bias (Wald 1943); mean deflation from censored data (statistics)

## Finding
S481 found compacted lessons had above-average degree (3.56 vs 3.23) and concluded compaction inflates K_avg. FALSIFIED. Restoring 109 compacted lessons (with 84 broken refs + 388 estimated outgoing) yields K_avg_full=3.3400 vs K_avg_raw=3.2449. Net effect: compaction DEFLATES K_avg by 0.095 (-2.9%). The S481 "correction" (PA 2.21x→1.38x) was an overcorrection.

**Mechanism**: High-degree compacted lessons contributed incoming edges to survivors. Removing them removes both: (a) the node (deflates N) and (b) their outgoing edges = other lessons' incoming edges. Since removed lessons had above-average degree, their edge contribution per node exceeds the denominator effect. Result: edges/N decreases.

## Rule
Compaction bias on K_avg is DEFLATIONARY, not inflationary. S481 PA correction (1.38x) should be reversed — raw PA (2.76x at S487) is the less-biased measure. For future compaction analysis: track edge REMOVAL separately from node removal. The asymmetry (edge loss > proportional node loss) is the bias direction indicator.

Falsified if: a controlled experiment (compact N=50 lessons, measure before/after) shows K_avg INCREASES after compaction.
