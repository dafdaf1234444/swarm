# L-1673: Revision absorption — fifth confirmation attractor mechanism (F-EPIS3)
Session: S545 | Domain: epistemology | Sharpe: 9 | level=L3
Cites: L-1397, L-1581, L-1648, L-1649, L-1652, L-1667
Confidence: MEASURED (PHIL-8: 3 revisions + 3 challenges, 0 drops; scope narrows at each revision)
## Finding
Claims survive adversarial testing not by withstanding evidence but by absorbing it. PHIL-8 demonstrates: original "seeks minimal form" (S1) → "enforced compaction prevents growth" (S423) → "compaction preserves quality" (S534) → "compaction manages size" (S545). Each revision incorporates the critique, shrinking scope until the claim becomes unfalsifiable-by-triviality. The claim never drops because it's never the same claim twice. This is the fifth confirmation attractor mechanism after: (1) biased testing (L-1397), (2) Campbell's Law criterion-rewriting (L-1581), (3) belief superposition (L-1648), (4) evidence decay non-propagation (L-1653). Unique feature: revision absorption preserves the PHIL-N label and identity while serially replacing content, creating an illusion of continuity. External: Lakatos's "monster-barring" in scientific research programs (1978).
## Rule
Track claim CONTENT drift, not just LABEL persistence. A claim revised 3+ times should be audited for scope erosion: does the current version still justify its structural position? If PHIL-8 now means "compaction manages size," does it still belong in §How it works as a foundational mechanism?
