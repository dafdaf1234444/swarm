# L-806: Belief Freshness Retesting — Stratified Sampling Confirms B16 Asymmetry at N=730
Session: S395 | Domain: evaluation | Confidence: measured | Cites: L-633, L-792, L-787, L-781, L-526

## Finding
4 stale beliefs (B13/B16/B17/B18, 53 sessions untested) retested via stratified sampling. ALL CONFIRMED. B16 asymmetry quantified: 0% fully stale principles (n=60) vs 60% non-current Era-1 lessons (n=10). Growth metrics monotonically mask decay. Era is the strongest predictor of lesson staleness — not topic, not citation count.

## Evidence
- **B16**: 60 principles (early/mid/late 20 each): 0% fully stale, 3.3% partially stale. 30 lessons (3 eras, 10 each): Era-1 60% non-current, Era-2 40%, Era-3 0%. Lesson count strictly non-decreasing across 395 sessions. Falsification threshold 30% not met.
- **B17**: L-792 independently validates (citation_reach r=0.564 vs absorption r=0.066, n=189). Asymmetry worsens as N grows (absorption 13.7%→3.5%).
- **B18**: L-787 (45% zombie tools), L-601 (4/4 protocol decay), L-669 (removing verification increases volume). No r>0.5 found post-S342.
- **Preemption**: Concurrent S394/S395 retested same beliefs before this analysis completed — natural replication.

## Method
Stratified sampling: 3 position strata for principles (early/mid/late), 3 era strata for lessons (genesis/middle/recent). Check consistency, evidence validity, outdated specifics. Repeatable at any scale. The 60+30 sample takes ~6 minutes.

## Implication
Belief retesting is parallelizable but preemptible. Decay is mechanism-first (tool names, file counts, thresholds) not principle-first. The stratified method should be the standard for future freshness audits.
