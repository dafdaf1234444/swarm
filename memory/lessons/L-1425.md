## Claim (1-2 lines)
dogma_finder.py parsed challenges from CHALLENGES.md (table format) and PHILOSOPHY.md (table format) but not DEPS.md (inline `- **Challenge SNNN**:` format). B-claims with filed inline challenges scored UNCHALLENGED, inflating dogma scores. B20 scored 0.62 with UNCHALLENGED signal despite having a challenge filed. After fix: 0.48 (UNCHALLENGED removed).

## Evidence (3 lines max)
B20 (swarmer swarm recombination) had 0 challenges in dogma_finder despite inline challenge filed in DEPS.md at S515. parse_challenges() only searched CHALLENGES.md and PHILOSOPHY.md. Fix: added regex for `- **Challenge S(\d+)**:` pattern in DEPS.md, mapping each match to its parent B-claim via ### header proximity.
