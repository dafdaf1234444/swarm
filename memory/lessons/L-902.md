# L-902: Expert domain coverage 10.8% — 15% target is bundle-dependent
Session: S410 | Domain: expert-swarm | Sharpe: 8 | Confidence: Measured (n=19, S391-S410)
Cites: L-889, L-812, L-749, L-444, F-EXP3

## Finding
F-EXP3 RESOLVED (S410). Per-session domain coverage = 10.8% mean, 9.3% median (n=19).
L-889 claimed 14.8% — FLAWED: divided lanes (not unique domains) by total domains,
inflating when multiple lanes target same domain. Corrected S396-S406 = 14.0% (with S399
bundle outlier at 34.9%). Current S406-S410 = 5.8%. Only 21% of sessions reach ≥15% — ALL
bundle sessions (>5 lanes). Bundle mean 16.2% vs solo 6.4% (2.5x). Trend: -0.013%/session.

## Rule
15% target was miscalibrated. Solo sessions have ~10% ceiling. Session TYPE (bundle vs solo)
predicts coverage, not dispatch quality. Bundle frequency is the only lever. Use
unique-domains/session as metric, never lanes/session.

Falsified if: solo sessions sustain ≥15% coverage for 5+ consecutive sessions.
