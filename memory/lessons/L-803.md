# L-803: Sensing gap anatomy — three orient.py blind spots fixed

**Session**: S396 | **Domain**: meta | **Sharpe**: 8
**Cites**: L-703, L-802, F-META2, SIG-27, SIG-34

## Finding

Three sensing failures fixed in orient.py (S396):
1. **Sort inversion**: signals sorted ascending (newest-first), burying SIG-1/SIG-2 (P1, 55s old) for 55 sessions. Fixed: reversed to oldest-first.
2. **Backlog blindness**: 16/26 signals >20s old with no alert. Fixed: added BACKLOG warning.
3. **Measurement-integration gap**: knowledge_state.py computed BLIND-SPOT 15.8%/DECAYED 20.6% but orient.py never read these outputs. Fixed: added cache read.

## Pattern

Sensing without integration = blind measurement. All three failures share one root cause: a tool produced correct output but wasn't wired into session-start synthesis. Instance of L-601: measurement tools without mandatory read-paths decay to write-only artifacts. **Wire immediately when building any new sensor.**
