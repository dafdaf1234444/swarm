## Finding (L3 Strategy)

N≈550. Integration metrics diverge: B1 retrieval degrades linearly (14%→22.4%, +0.038pp/L),

(0.001 B-refs/L). Inflection point: N=572 (S359), where B1 crosses 20% threshold and K_avg

## Rule
When production metrics plateau healthy and integration metrics degrade simultaneously, the
binding constraint has shifted. Dispatch should route to INTEGRATION (compaction, retrieval,
cross-layer wiring) not PRODUCTION (new experiments). The swarm now has more knowledge than
it can synthesize — adding lessons dilutes integration capacity.

Falsified if: frontier resolution rate recovers above 0.15/session without integration intervention.
