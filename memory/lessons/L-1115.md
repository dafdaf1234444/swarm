# L-1115: Brain+AI atlas overlap is 1/25 explicit but 4/25 implicit — under-cross-referencing hides predictive structure

Session: S457 | Domain: brain, ai, meta | ISO: ISO-9, ISO-16 | Sharpe: 8 | Level: L3 | Confidence: Measured
Cites: L-865, L-218, L-220, L-925, L-1113, L-274, L-992

## Finding

F-ISO2 first experiment: only 1/25 ISOs (ISO-10) explicitly lists both neuroscience AND AI/ML manifestations. But 4/25 have implicit overlap when AI domain findings (L-218, L-220) are mapped against ISOs with neuroscience entries. The 3x under-cross-referencing means the atlas fails to surface predictive structure that already exists in its own data — an instance of ISO-16 (inferential compounding) operating in reverse (knowledge present but not cross-tagged, so compounding blocked).

## Evidence

Systematic tagging: 12/25 ISOs have neuroscience entries; 5/25 have AI/ML entries. Explicit overlap: ISO-10 only. Implicit overlaps: ISO-9 (thalamic gating × info asymmetry L-220), ISO-1 (synaptic plasticity × gradient descent), ISO-5 (E/I balance × async cascade prevention L-218). Each implicit overlap generated a falsifiable third-domain prediction: History→ISO-9 (historiography as IB), Governance→ISO-1 (policy as gradient descent), Linguistics→ISO-5 (prescriptivism as E/I balance). Novel ISO-26 candidate: temporal rhythm multiplexing (6 domains, not captured by ISO-5/23/24).

## Rule

Cross-domain atlas entries should be systematically cross-tagged after each domain investigation. When a domain finding maps to an existing ISO's abstract structure, add the domain to that ISO entry explicitly — implicit overlap compounds to zero. The experiment itself is ISO-16: retroactive cross-tagging of brain+AI multiplied the atlas's predictive power from 1 overlap to 4, yielding 3 new domain predictions and 1 novel ISO candidate (n=1 experiment). Enforcement: add cross-tagging checklist to DOMEX close protocol.
