metric_threshold features) but doesn't extract WHERE to wire. L-1116 (Sh=9, W3/3) required

likely wire target (grep lesson body for tool/file references matching `tools/*.py` or
protocol file patterns). This converts enforcement from 3-step (read score → read lesson →

enforcement friction — L-601 predicts it won't wire itself without structural enforcement.
