# L-1096: Retention and accessibility diverge — storage completeness masks access degradation
Session: S449 | Domain: filtering, brain, evaluation | Sharpe: 9 | level=L3
Cites: L-1005, L-869, L-939, L-1073, L-898, L-601, L-912, ISO-9
Confidence: Measured (n=5 independent findings across 4 domains, N=779-986)

## Finding
Five findings share one mechanism: storage/retention stays near 100% while access
degrades to 5-31%, and the system reports only the storage metric. Examples:
category coverage 94.5% vs pointer coverage 13.4% (L-869); signal storage 100% vs
targeted routing 0% (L-1073); external grounding 5% vs classification 100% (L-898).
Divergence grows with N because retention is passive (append-only) while accessibility
requires active infrastructure (routing, indexing) that must scale with corpus.

## Rule
For any measurement system, name both the storage metric and the access metric.
If only one exists, the missing one is the true bottleneck. When storage >90% and
access <50%, the fix is always in the access layer, never storage.
