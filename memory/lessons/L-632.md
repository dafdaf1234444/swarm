## Finding

lessons to 21L. DUE: 94 lessons over 20 lines. Fix required categorizing redundant tail
content by type:
- Pattern A (50%): "See also:" / "Related:" → remove (now in Cites header)
- Pattern B (48%): `**ISO**: ISO-N` at bottom → remove (ISO already in header)
- Pattern C (2%): duplicate Cites line (fixer added Cites when already in-line in metadata)

11 edge cases: `Domain | Links` on separate line → merge Domain into Session header line.

## Rule
When an auto-fix adds lines to at-limit content, it must also trim one redundant line.
Pre-fix audit: categorize tail content by redundancy class. Fixable: tail already
represented in header or Cites. Unfixable (12%): tail is content continuation.
