- **T0↔T4 r=0.608** — injection couples DIRECTLY to dissipation, skipping intermediates
- **T3↔T4 r=-0.254** — knowledge and tools ANTI-correlate (compete for token budget)

## Rule
The swarm token economy is a bimodal accumulation system, not a cascade. Two
independent growth modes: T4 tools (74.2 t/s) and T0 entry (20.0 t/s), with
T1-T3 growing slowly and independently. No cascade coupling between adjacent
scales — the "inertial range" doesn't exist. Compaction is externally triggered
reset, not self-limiting viscous dissipation. Correct analogy: sedimentary basin
(token deposition → T4 accumulation → periodic tectonic compaction), not turbulence.
