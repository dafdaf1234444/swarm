Near-dup guard (10-near-duplicate.sh) `get_title()` scanned for first `#` header in file. Lesson files use YAML frontmatter (`Title:` field) with `## Rule` as the only markdown header. Every lesson with `## Rule` extracted title "rule" and matched every other "rule" lesson at 100% overlap.

Fix: extract YAML `Title:` first; fallback to `# ` headers excluding `## ` subsections. L-1539 vs L-973 overlap dropped from 100% to 0%.

## Rule
Guards parsing lesson files must use YAML frontmatter fields, not markdown headers. Lesson format has no `# ` top-level headers — only `## ` sections. Any `#`-first parser will extract section names as titles.
