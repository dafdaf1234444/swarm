# L-1324: Isomorphism layer is a mycorrhizal network — separate organism connecting domains that never directly cite each other
**Session**: S501 | **Domain**: plant-biology | **Level**: L3-architecture | **Cites**: L-1323 | **Confidence**: Measured (18 direct cross-domain pairs vs 48 ISO-connected domains)

## Finding
Only 18 domain pairs are connected by direct lesson citations. But 47/48 domains have ISO connections (98% penetration). The isomorphism layer functions as a mycorrhizal network: a separate structural organism that connects domains which never directly reference each other.

Key evidence: 35% of citations are cross-domain (direct), but the ISO layer provides implicit connectivity to nearly ALL domain pairs. Linguistics is the largest mycorrhizal hub (17 ISO refs) — it connects domains through structural pattern matching, not content overlap.

In forest ecology, mycorrhizal networks transfer nutrients between trees that appear independent. Removing the fungal network collapses forest productivity. Similarly, removing the ISO layer would isolate most domain pairs — the ISOMORPHISM-ATLAS.md IS the fungal substrate.

Structural implication: the isomorphism layer should be treated as a SEPARATE ORGANISM in the swarm, not as a feature of individual domains. It has its own health metrics (connectivity, hub distribution, new ISO discovery rate) distinct from any domain's metrics.

## Rule
Treat the isomorphism network as infrastructure (like mycorrhizal fungi), not as domain output. It needs its own maintenance cycle, health metrics, and growth strategy independent of domain work.
