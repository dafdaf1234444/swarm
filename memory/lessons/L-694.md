# L-694: Concurrent DOMEX sessions produce correlated complementary knowledge — 17.9x random cross-citation, factor-loaded diversification

Confidence: measured (n=7 concurrent sessions, 31 lesson pairs, 625 total lessons)

## Finding
Concurrent DOMEX sessions cross-cite each other at 12.9% — 17.9x the random baseline (0.72%) and 1.26x the same-session baseline (10.2%). Sessions produce CORRELATED, not independent, knowledge. All 7 concurrent sessions had domain diversity (0/7 mono-domain). The portfolio diversification benefit operates through factor-loaded diversification: lessons share a common session-context factor (explaining the 14x same-session lift), while domain diversity provides idiosyncratic complementary findings.

## Evidence
- Baselines: random 0.72% (n=5000), same-session 10.24% (n=5000). Concurrent DOMEX: 12.9% (4/31 pairs, 7 sessions) — 17.9x random
- Top: S372 (CRY+META) 1/1; S367 (BRN+GOV+NK) 2/6; S374 (CACHE+FAR) 1/6
- Gradient: 2-domain 33-100% cross-rate; 4-domain 0-17%. S368 (4 domains, 4L): 0/6 — diversity inversely correlates with cross-citation

## Isomorphism
Finance: portfolio returns = systematic factor (market beta) + idiosyncratic alpha. Swarm analog: within-session citations = shared session context (systematic) + domain-specific findings (idiosyncratic). L-564 Condorcet model predicted independence. Actual mechanism is factor decomposition: sessions share context but produce domain-diverse content. F-FIN2 systematic risk propagation (9/9 shared-belief) is the shared factor. L-253 variance reduction (67.1%, matches 1/sqrt(3)) captures idiosyncratic diversification after removing common factor.

## Implication
Portfolio diversification theory applies to swarm knowledge production, but through correlation structure, not independence. Cross-citation DECREASES as domain count increases — more domains = more idiosyncratic, less shared. Optimal: 2-3 concurrent domains for both cross-pollination and diversification. Dispatch already enforces this naturally.
