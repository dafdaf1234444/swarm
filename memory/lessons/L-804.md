# L-804: Swarm science is systematic self-measurement, not discovery — confirmation machine diagnosis
Session: S396 | Domain: meta | Sharpe: 9 | Confidence: Measured (n=683 experiments, n=388 sessions, n=725 lessons)
Cites: L-787, L-797, L-601, L-633, L-792, L-778

## Finding
Human directive SIG-36: "swarm science has to improve for the swarm." Audit reveals: swarm optimizes for confirmation speed over discovery depth. It measures itself systematically but calls monitoring science.

## Evidence — Six weaknesses
1. **Confirmation dominance**: 58:1 confirm-to-discover ratio (L-787). Frontiers opened as pre-answered questions.
2. **Self-referentiality**: 78% of work studies the swarm itself. No external validation.
3. **Zero falsification**: 0 DROPPED challenges in 388 sessions. Challenge mechanism = confirmation loop.
4. **Monitoring as experiment**: 20% of 683 experiments have genuine controls. 70% lack significance testing. Thresholds set post-hoc.
5. **Closed epistemic loop**: Meta-prediction accuracy 33% — worst domain, most studied (42.5% of all work).
6. **Dead weight**: 32.1% lessons never cited. 40/44 domains zero attention. 45 zombie tools.

## Fix (P-243)
Pre-registration + adversarial lanes + external validation + significance testing + numeric predictions. L-601: voluntary rigor decays; only protocol-enforced standards persist.

Falsified if: pre-registration + adversarial lanes produce no change in discover/confirm ratio after 20 sessions.
