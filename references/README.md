# References

`references/` stores curated external-source material used by swarm work.

## What belongs here
- Source notes and distilled citations (`.md`)
- Structured source metadata (`.json`)
- Lightweight tabular/bibliographic references (`.csv`, `.tsv`, `.bib`)

## File-type policy
- Allowed: `.md`, `.json`, `.txt`, `.csv`, `.tsv`, `.bib`, `.yml`, `.yaml`
- Exceptions: `README.md`, `.gitkeep`

## Suggested layout
- `references/<domain>/...`
- `references/<domain>/<topic>-s<session>.md`

Prefer linking to external primary sources instead of storing large raw documents in-repo.
